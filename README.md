# nesemu [WIP]
NES emulator in modern c++

## Dependencies:
- cmake (>= 3.12.1)
- sdl2 (>= 2.0.8)
- sdl2_ttf (>= 2.0.14)

## After clone:
- install dependencies. If you have apt, run: `$ sudo make install-libraries`
- build cmake: `$ make build`

## Compile:
`$ make compile`

## Run:
`$ make run rom=/path/to/rom.ines`
